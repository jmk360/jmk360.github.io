## 문자열 입력받아서 그대로 출력

- 예제코드
  
```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main(void) {
	char a[20];
	scanf("%s", a);
	printf("%s\n", a);
	system("pause");
	return 0;
}
```

- 출력결과
  - 영문을 입력했을 경우 정상 출력
  - 한글을 입력했을 경우 출력이 깨지는 현상이 생겼다.
