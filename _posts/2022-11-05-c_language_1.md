## 하나의 정수를 입력받아 한 자리 씩 끊어서 변수에 입력하기

- 예제코드
  
```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main(void) {
	int a, b, c;
	scanf("%1d%1d%1d", &a, &b, &c);
	printf("%d, %d, %d\n", a, b, c);
	system("pause");
	return 0;
```
